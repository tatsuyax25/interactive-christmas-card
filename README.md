# Interactive Christmas Card

A festive, animated Christmas card built with HTML5 Canvas and JavaScript featuring interactive characters, falling snow, and twinkling lights.

## Features

- **Animated Characters**: Switch between Santa, Snowman, and Reindeer with smooth animations
- **Dynamic Weather**: Realistic falling snow particles
- **Christmas Lights**: Twinkling string lights with color-changing effects
- **Background Music**: Holiday music with play/pause controls
- **Responsive Design**: Works on desktop and mobile devices

## Technologies Used

- HTML5 Canvas for animations
- Vanilla JavaScript for interactivity
- CSS3 for styling and responsive design
- Audio API for music playback

## Getting Started

1. Clone or download the project
2. Open `index.html` in a web browser
3. Click the navigation buttons to switch characters
4. Use the music button to play/pause holiday music

## Project Structure


## Browser Compatibility

Works in all modern browsers that support HTML5 Canvas and Audio APIs.
