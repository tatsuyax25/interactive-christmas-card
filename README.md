# Interactive Christmas Card

A festive, animated Christmas card built with HTML5 Canvas and JavaScript featuring interactive characters, falling snow, and twinkling lights.

## Screenshot

![Interactive Christmas Card Screenshot](https://imgur.com/rMETZj3)

## Features

- **Animated Characters**: Santa, Snowman, and Reindeer with smooth waving animations
- **Santa's House**: Cozy house with smoking chimney, wreath, and twinkling lights
- **Christmas Tree**: Decorated tree with twinkling ornaments and gold star
- **Dynamic Weather**: Realistic falling snow particles
- **Christmas Lights**: Twinkling string lights across the top
- **Background Music**: Holiday music with play/pause controls
- **Responsive Design**: Works on desktop and mobile devices
- **Animated Title**: Color-changing title text mimicking Christmas lights

## Technologies Used

- HTML5 Canvas for animations
- Vanilla JavaScript for interactivity
- CSS3 for styling and animations
- Google Fonts (Mountains of Christmas)
- Audio API for music playback

## Getting Started

1. Clone or download the project
2. Open `index.html` in a web browser
3. Use the music button to play/pause holiday music
4. Enjoy the animated Christmas scene!
